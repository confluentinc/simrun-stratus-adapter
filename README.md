# simrun-stratus-adapter

A [SimRun](https://github.com/IBM/simrun) pack that mirrors attack techniques from [Stratus Red Team](https://github.com/DataDog/stratus-red-team) by Datadog. It adapts Stratus cloud attack simulations (mapped to MITRE ATT&CK) into the SimRun pack framework for automated security testing and validation.
